# 股票数据可视化分析系统 v2.0

Stock Data Visualization Analysis System

---

## 项目简介

基于 Python + Gradio 构建的股票数据分析 Web 应用，支持实时 Yahoo Finance 数据获取和本地模拟数据两种模式。提供收益率分析、涨跌动量指标分析、股息分析三大核心模块，通过可视化图表直观展示多维度股票对比。

---

## 功能特性

### 核心分析模块

| 模块 | 功能说明 |
|------|---------|
| **收益率概览** | 总收益率、年化收益率、起始价/最新价对比 |
| **涨跌分析** | 波动率、最大回撤、均线偏离度（20日/60日）、近1月/3月/6月/1年涨跌幅 |
| **股息分析** | 年化股息、股息率、派息次数、最近股息金额 |

### 可视化图表

- 总收益率对比柱状图
- 价格走势归一化折线图
- 各期涨跌幅分组对比图
- 波动率 vs 最大回撤对比图
- 股息率对比图
- 均线偏离度对比图

### 数据源支持

| 数据源 | 说明 |
|--------|------|
| **Yahoo Finance (实时)** | 联网获取真实股票行情，支持美股/A股 |
| **模拟数据 (免联网)** | 基于股票代码生成确定性模拟数据，无需网络 |

## 使用指南

### 1. 输入股票代码

在"股票代码"输入框中填写股票代码，多个代码用逗号分隔：

```
AAPL,MSFT,TSLA,NVDA
```

| 市场 | 代码示例 | 说明 |
|------|---------|------|
| 美股 | AAPL, MSFT, TSLA, NVDA, JPM, KO | 直接输入代码 |
| A股 | 000001.SZ, 600519.SS | 需加后缀 .SZ / .SS |

### 2. 设置日期范围

- **开始日期**：分析起始时间，格式 `YYYY-MM-DD`
- **结束日期**：分析截止时间，格式 `YYYY-MM-DD`

### 3. 选择数据源

| 模式 | 适用场景 |
|------|---------|
| **模拟数据 (免联网)** | 无网络环境、快速测试、避免API限流 |
| **Yahoo Finance (实时)** | 需要真实行情数据 |

### 4. 查看结果

点击"开始分析"后，在右侧查看：
- **分析摘要**：关键指标汇总
- **收益率概览**（Tab）：收益率数据表
- **涨跌分析**（Tab）：动量指标数据表
- **股息分析**（Tab）：股息数据表
- **可视化图表**：6维度的对比图表

## 核心算法说明

### 收益率计算

```
总收益率 = (最新价 - 起始价) / 起始价 * 100%

年化收益率 = ((最新价 / 起始价) ^ (1 / 年数) - 1) * 100%
```

### 波动率

```
日收益率 = (当日收盘价 - 前日收盘价) / 前日收盘价
年化波动率 = 标准差(日收益率) * sqrt(252) * 100%
```

### 最大回撤

```
回撤 = (当前价格 - 历史最高点) / 历史最高点
最大回撤 = min(回撤序列) * 100%
```

### 均线偏离

```
N日均线偏离 = (当前价 - N日均价) / N日均价 * 100%
```

## 注意事项

1. **Yahoo Finance 限流**：频繁请求可能触发限流，如遇错误请切换至"模拟数据"模式
2. **日期格式**：请严格使用 `YYYY-MM-DD` 格式（如 2020-01-01）
3. **公开链接**：`share=True` 生成的公开链接有效期约72小时，仅供临时分享
4. **模拟数据特性**：同一代码每次生成相同的数据序列（确定性随机），便于复现结果
